================================================================================
🔧 DÉPANNAGE
================================================================================

Problème: "Le terme '.\venv_asl3d\Scripts\Activate.ps1' n'est pas reconnu"
Solution: 
  Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
  .\venv_asl3d\Scripts\Activate.ps1

Problème: "No module named 'flask'" après activation
Solution:
  pip list  # Vérifier l'installation
  pip install Flask  # Réinstaller si besoin

Problème: Port 5000 déjà utilisé
Solution:
  Terminer le processus Python existant
  Ou modifier le port dans app.py ligne: app.run(port=5001)
