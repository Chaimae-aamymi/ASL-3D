FONCTIONNALITÉS CLÉS
================================================================================

1. CLASSIFICATION (MobileNetV2)
   ├─ Input: Image 224×224 RGB
   ├─ Base: MobileNetV2 pré-entraîné (ImageNet)
   ├─ Poids congelés: Transfer learning activé
   ├─ Couches custom: Dense(256) → Dense(128) → Dense(6)
   ├─ Régularisation: BatchNorm + Dropout
   ├─ Output: 6 probabilités [fissures, humidite, erosion, champignons, decoloration, effritement]
   └─ Performance: 85-95% accuracy attendue

2. SEGMENTATION (U-Net)
   ├─ Input: Image 256×256 RGB
   ├─ Architecture: Encodeur-Décodeur symétrique
   ├─ Skip connections: Conservation info spatiale
   ├─ Bottleneck: Conv2D(256)
   ├─ Output: Masque binaire 256×256
   └─ Performance: 0.80-0.95 Dice score attendu

3. DÉTECTION INTÉGRÉE
   ├─ Prétraitement automatique
   ├─ Classification + Segmentation en pipeline
   ├─ Fusion des résultats
   ├─ Extraction boîtes englobantes
   ├─ Scores de confiance par détection
   └─ Temps: <1s (GPU) ou 1-2s (CPU)

4. ENTRAÎNEMENT PERSONNALISÉ
   ├─ Méthode train_models() complète
   ├─ Support data augmentation
   ├─ Callbacks (EarlyStopping, Checkpoints)
   ├─ Sauvegarde/chargement (.h5)
   ├─ Compatible GPU NVIDIA
   └─ Amélioration progressive possible

5. MODE FALLBACK INTELLIGENT
   ├─ Fonctionne sans modèles entraînés
   ├─ Filtres traditionnels automatiques
   ├─ Transition transparente
   ├─ Garantit opérationnalité
   └─ Prêt pour amélioration


================================================================================
ARCHITECTURE DÉTAILLÉE
================================================================================

Pipeline complet:

Image (quelque résolution)
    ↓
_preprocess_image()
├─ Conversion RGB si grayscale
├─ Redimensionnement 224×224
└─ Normalisation [0, 1]
    ↓
_classify_degradations()
├─ MobileNetV2 forward pass
├─ Dense(256) ReLU + BatchNorm
├─ Dense(128) ReLU + BatchNorm
├─ Dense(6) Softmax
└─ Output: probas [p_fissures, p_humidite, ...]
    ↓
_segment_degradations()
├─ Redimensionnement 256×256
├─ U-Net forward pass
├─ Encodeur: Conv×3 → MaxPool×3
├─ Bottleneck: Conv2D(256)
├─ Décodeur: UpSample×3 → Concat → Conv×3
└─ Output: masque binaire
    ↓
_merge_predictions()
├─ Fusion classification + segmentation
├─ Détection contours masque
├─ Attribution types aux zones
├─ Calcul boîtes englobantes
└─ Output: [{'type': ..., 'location': ..., 'confidence': ...}, ...]
    ↓
Dégradations détectées


================================================================================
UTILISATION
================================================================================

SANS ENTRAÎNEMENT (Mode fallback):
```python
from degradation_detector import DegradationDetector
detector = DegradationDetector()  # Crée modèles par défaut
degradations = detector.detect(image)  # Utilise filtres fallback
```

AVEC MODÈLES ENTRAÎNÉS:
```python
detector = DegradationDetector()
detector.load_models('models/')  # Charge modèles entraînés
degradations = detector.detect(image)
severity = detector.calculate_severity(degradations)
recommendations = detector.get_recommendations(degradations)
```

ENTRAÎNER MODÈLES:
```python
# Voir GUIDE_ENTRAINEMENT.txt pour détails
detector.train_models(X_train, y_train, epochs=50)
detector.save_models('models/')
```
