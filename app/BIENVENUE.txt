🧠 ARCHITECTURE CNN
═══════════════════════════════════════════════════════════════════════════════

MODÈLE 1 - CLASSIFICATION (MobileNetV2):
  
  Input (224×224 RGB)
      ↓
  MobileNetV2 pré-entraîné (ImageNet)
  ├─ 13 bottleneck blocks
  ├─ Poids congelés (Transfer Learning)
  └─ Extraction features
      ↓
  GlobalAveragePooling2D
      ↓
  Dense(256, ReLU) + BatchNorm + Dropout(0.5)
      ↓
  Dense(128, ReLU) + BatchNorm + Dropout(0.3)
      ↓
  Dense(6, Softmax)
      ↓
  Output: [p_fissures, p_humidite, p_erosion, p_champignons, p_decoloration, p_effritement]

  Performance:
    - Speed: 50ms (GPU) / 300ms (CPU)
    - Accuracy: 85-95% (après entraînement)
    - Taille: 40 MB
    - Paramètres: 3.5M


MODÈLE 2 - SEGMENTATION (U-Net):
  
  Input (256×256 RGB)
      ↓
  ENCODEUR:
  Conv2D(32) → MaxPool     [256→128]
  Conv2D(64) → MaxPool     [128→64]
  Conv2D(128) → MaxPool    [64→32]
      ↓
  BOTTLENECK:
  Conv2D(256)
      ↓
  DÉCODEUR:
  UpSample + Concat + Conv2D(128)  [32→64]
  UpSample + Concat + Conv2D(64)   [64→128]
  UpSample + Concat + Conv2D(32)   [128→256]
      ↓
  Conv2D(1, Sigmoid)
      ↓
  Output: Masque binaire (256×256)

  Performance:
    - Speed: 50ms (GPU) / 400ms (CPU)
    - Dice Score: 0.80-0.95 (après entraînement)
    - Taille: 15 MB
    - Paramètres: 1.9M

🎯 UTILISATION (3 CAS)
═══════════════════════════════════════════════════════════════════════════════

CAS 1 - SANS ENTRAÎNEMENT (Mode fallback):
  ```python
  from degradation_detector import DegradationDetector
  detector = DegradationDetector()
  degradations = detector.detect(image)
  # Utilise filtres image automatiquement
  ```

CAS 2 - AVEC MODÈLES ENTRAÎNÉS:
  ```python
  detector = DegradationDetector()
  detector.load_models('models/')
  degradations = detector.detect(image)
  # Haute confiance!
  ```

CAS 3 - ENTRAÎNER VOS MODÈLES:
  ```python
  detector = DegradationDetector()
  detector.train_models(X_train, y_train, epochs=50)
  detector.save_models('models/')
  # Prêt pour production!
  ```
