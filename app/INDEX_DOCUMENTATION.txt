✅ degradation_detector.py (440 lignes) ⭐ NOUVEAU DEEP LEARNING
   Description: Détecteur IA avec modèles CNN (MobileNetV2 + U-Net)
   Contient:
     ├─ DegradationDetector class
     ├─ build_models(): Construit MobileNetV2 + U-Net
     ├─ detect(): Pipeline complet détection
     ├─ _classify_degradations(): Classification CNN
     ├─ _segment_degradations(): Segmentation U-Net
     ├─ _merge_predictions(): Fusion résultats
     ├─ train_models(): Entraînement personnalisé
     ├─ save_models() / load_models(): Gestion fichiers
     ├─ visualize(): Création visualisations
     ├─ calculate_severity(): Sévérité globale
     ├─ get_recommendations(): Recommandations
     ├─ Mode fallback: Filtres image si pas de modèles
     └─ Support GPU/CPU transparent
   API Key Methods:
     - detect(image_array) → [degradations]
     - train_models(X, y, epochs) → entraînement
     - save_models(path) → sauvegarde
     - load_models(path) → chargement

✅ reconstruction_engine.py (450+ lignes)
   Description: Moteur de reconstruction 3D
   Contient:
     ├─ ReconstructionEngine class
     ├─ reconstruct(): Créer modèle 3D
     ├─ _generate_depth_map(): Conversion 2D→3D
     ├─ _create_mesh(): Génération maillage
     ├─ save_model(): Export OBJ
     ├─ generate_restoration_plan(): Plan détaillé
     └─ Estimations coûts/délais/sécurité
   Utilisation: Appelé par app.py /api/reconstruct-3d
