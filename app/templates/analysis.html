Guide des Types de Dégradation

Fissures

Sévérité : Haute

Description : Ruptures dans la structure, peuvent indiquer des problèmes structurels.

Action : Évaluation urgente requise

Humidité

Sévérité : Moyenne

Description : Infiltrations d'eau, effloresences salines, zones humides.

Action : Diagnostic de source et traitement

Érosion

Sévérité : Moyenne

Description : Usure progressive de la surface due aux agents climatiques.

Action : Protection et retraitement

Champignons

Sévérité : Haute

Description : Moisissures, lichens, végétation pathogène.

Action : Traitement immédiat

Décoloration

Sévérité : Basse

Description : Perte de couleur, altération esthétique.

Action : Nettoyage et protection

Effritement

Sévérité : Moyenne

Description : Désagrégation du matériau, perte de cohésion.

Action : Consolidation et finition
