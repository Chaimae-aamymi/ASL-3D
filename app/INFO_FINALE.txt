🚀 DÉMARRAGE RAPIDE
═════════════════════════════════════════════════════════════════════════════════

1. Ouvrir terminal/PowerShell

2. Se placer dans le dossier:
   cd c:\Users\HP\pfe\app

3. Installer les dépendances:
   pip install -r requirements.txt

4. Lancer l'application:
   python run.py

5. Ouvrir dans navigateur:
   http://127.0.0.1:5000
