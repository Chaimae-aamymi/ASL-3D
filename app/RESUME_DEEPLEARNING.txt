================================================================================
NOUVELLES FONCTIONNALITÉS
================================================================================

1. Classification Deep Learning
   ├─ MobileNetV2 pré-entraîné (ImageNet)
   ├─ Transfer Learning pour les 6 types de dégradations
   ├─ Scores de confiance: 0.0 - 1.0
   └─ Batch normalization + Dropout pour régularisation

2. Segmentation Sémantique
   ├─ U-Net pour localisation pixel-parfaite
   ├─ Masque binaire des zones dégradées
   ├─ Skip connections pour conservation spatiale
   └─ Capable de détecter géométries complexes

3. Pipeline Intégré
   ├─ Prétraitement automatique
   ├─ Classification + Segmentation en séquence
   ├─ Fusion intelligente des résultats
   ├─ Extraction de boîtes englobantes
   └─ Gestion erreurs gracieuse

4. Entraînement Personnalisé
   ├─ Méthode train_models() complète
   ├─ Support data augmentation
   ├─ Callbacks (EarlyStopping, ModelCheckpoint)
   ├─ Sauvegarde/chargement des modèles (.h5)
   └─ Compatible GPU (CUDA/cuDNN)

5. Mode Fallback Intelligent
   ├─ Si modèles non entraînés → filtres traditionnels
   ├─ Garantit fonctionnement même sans données
   ├─ Transition transparente
   └─ Prêt pour amélioration progressive


================================================================================
ARCHITECTURE TECHNIQUE
================================================================================

Modèle de Classification:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Input (224×224×3 RGB)
    ↓
MobileNetV2 pré-entraîné
    ├─ 13 bottleneck blocks
    ├─ Poids ImageNet congelés
    └─ Extraction features robustes
    ↓
GlobalAveragePooling2D
    ↓
Dense(256, ReLU) + BatchNorm + Dropout(0.5)
    ↓
Dense(128, ReLU) + BatchNorm + Dropout(0.3)
    ↓
Dense(6, Softmax)  [fissures, humidite, erosion, champignons, decoloration, effritement]
    ↓
Output: Vecteur probabilités [0-1]


Modèle de Segmentation:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Input (256×256×3)
    ↓
ENCODEUR:
  Conv2D(32) → MaxPool    [256×256 → 128×128]
  Conv2D(64) → MaxPool    [128×128 → 64×64]
  Conv2D(128) → MaxPool   [64×64 → 32×32]
    ↓
BOTTLENECK:
  Conv2D(256)
    ↓
DÉCODEUR:
  UpSample + Concat + Conv2D(128)   [32×32 → 64×64]
  UpSample + Concat + Conv2D(64)    [64×64 → 128×128]
  UpSample + Concat + Conv2D(32)    [128×128 → 256×256]
    ↓
Conv2D(1, Sigmoid)
    ↓
Output: Masque binaire (256×256)

================================================================================
COMMENT UTILISER
================================================================================

Sans entraînement (Mode fallback):
```python
from degradation_detector import DegradationDetector
import cv2

detector = DegradationDetector()
image = cv2.imread('building.jpg')
degradations = detector.detect(image)
# Utilise filtres traditionnels automatiquement
```

Avec modèles pré-entraînés:
```python
detector = DegradationDetector()
detector.load_models('models/')  # Charger modèles entraînés
degradations = detector.detect(image)
```

Entraîner vos modèles:
```python
# Voir GUIDE_ENTRAINEMENT.txt pour détails complets
detector.train_models(X_train, y_train, epochs=50)
detector.save_models('models/')
```
