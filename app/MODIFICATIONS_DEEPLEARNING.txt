🆕 NOUVELLES CAPACITÉS
================================================================================

CLASSIFICATION (MobileNetV2):
  ├─ Input: 224×224 RGB images
  ├─ Base: MobileNetV2 pré-entraîné (ImageNet)
  ├─ Couches custom: Dense(256) + Dense(128) + Dense(6)
  ├─ Output: 6 probabilités (confiance pour chaque type)
  ├─ Accuracy attendue: 85-95%
  └─ Speed: 50ms (GPU) / 300ms (CPU)

SEGMENTATION (U-Net):
  ├─ Input: 256×256 RGB images
  ├─ Architecture: Encodeur-Décodeur + Skip connections
  ├─ Output: Masque binaire de zones dégradées
  ├─ Dice score attendu: 0.80-0.95
  └─ Speed: 50ms (GPU) / 400ms (CPU)

SCORES DE CONFIANCE:
  ├─ Chaque détection: 0.0 - 1.0
  ├─ Interprétation:
  │   ├─ >0.8: Haute confiance
  │   ├─ 0.6-0.8: Confiance moyenne
  │   ├─ 0.3-0.6: À valider
  │   └─ <0.3: Filtré
  └─ Recommandations adaptées

ENTRAÎNEMENT PERSONNALISÉ:
  ├─ train_models(X, y, epochs, batch_size)
  ├─ Support data augmentation
  ├─ Callbacks (EarlyStopping, ModelCheckpoint)
  ├─ Sauvegarde modèles: save_models(path)
  ├─ Chargement: load_models(path)
  └─ Amélioration progressive possible


================================================================================
⚙️ ARCHITECTURE TECHNIQUE
================================================================================

PIPELINE COMPLET:

Image → Prétraitement
   ↓
Classification (MobileNetV2)
   ├─ Identifie types présents
   └─ Output: [p_fissures, p_humidite, ...]
   ↓
Segmentation (U-Net)
   ├─ Localise zones dégradées
   └─ Output: Masque binaire
   ↓
Fusion (merge_predictions)
   ├─ Combine classification + segmentation
   ├─ Détecte contours
   └─ Attribution types → zones
   ↓
Résultat → [{'type': ..., 'location': {...}, 'confidence': ...}]


MODÈLES INCLUS:

MobileNetV2 (Classification):
  ├─ Pré-entraîné: ImageNet
  ├─ Couches: Base model (1280) → Dense(256) → Dense(128) → Dense(6)
  ├─ Poids congelés: Transfer learning activé
  ├─ Régularisation: BatchNorm + Dropout
  ├─ Taille: 40 MB
  └─ Paramètres: 3.5M

U-Net (Segmentation):
  ├─ Encodeur: Conv2D(32→64→128) + MaxPool
  ├─ Bottleneck: Conv2D(256)
  ├─ Décodeur: UpSample + Concat + Conv2D(128→64→32)
  ├─ Output: Conv2D(1, Sigmoid)
  ├─ Taille: 15 MB
  └─ Paramètres: 1.9M
